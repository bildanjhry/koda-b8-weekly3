# McDonalds Clone

Program clone McDonalds menggunakan GO sebagai bahasa pemogramannya, dimana program ini menawarkan pesan makanan/minuman McDonalds.

### Layanan yang ditawarkan:
- Browse Produk
- Lihat Keranjang
- Kurangi Produk dikeranjang
- Checkout
- PIlih opsi checkout Cash/Qris

### Binary App (Linux) 
v1.0.9 :
[mcd_clone-linux-amd64](https://github.com/bildanjhry/koda-b8-weekly3/releases/download/v1.0.9/mcd_clone-linux-amd64)

### Ubah permission:
```shell
chmod +x ./mcd_clone-linux-amd64
```
### Jalankan:
```shell
./mcd_clone-linux-amd64
```
